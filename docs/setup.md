# Setup Guide

## Frontend

```bash
cd frontend
npm install
npm run dev

```
Runs at:
http://localhost:3000

## Backend
 ```bash
 cd backend
npm install
npm run dev
```
Runs at:
http://localhost:5000

# NOTES

- No authentication is required

- No database is required

- Features are intentionally minimal
---
